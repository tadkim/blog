USEABLE :: CDN Achive

이 페이지는 여러 자바스크립트 라이브러리의 CDN을 모아두는 곳으로, 필요시 빠르게 참조하기 위해 작성하였다.

  • 부트스트랩(Bootstarp)
    • CSS 최소화 : <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.2/css/bootstrap.min.css">
    • CSS (부가적인 테마포함) : <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.2/css/bootstrap-theme.min.css">
    • JS (최소화) : <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.2/js/bootstrap.min.js"></script>
