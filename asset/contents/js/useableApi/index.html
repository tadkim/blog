Array

var arrays = [ 3, "디자인", ["시각", "산업", "패션"], "개발자", 2016]; 위 배열의 색인은 다음과 같다. * 3 : [0] * "디자인" : [1] * "시각" : [2][0] * "산업" : [2][1] * "패션" : [2][2] * "디자이너" : [3] * 2016 : [4]

배열 다루기 배열은 Property와 Method로 다룬다. * Property : 보통 배열에 대한 정보를 알려준다. * Method : 배열을 변경하거나 새로운 배열을 반환하는데 사용한다.

Array - 자주사용하는 API

.push - (끝부분)배열 원소 추가하기 : var arrays = []; arrays.push("추가할 원소") .unshift - (시작부분)배열 원소 추가하기 : var arrays = []; arrays.unshift("추가할 원소") .pop - (끝부분)배열 원소 제거하기 : var arrays = ["산업", "패션", "시각"]; arrays.pop() 해당 배열의 마지막 원소가 제거된다. .shift - (시작부분)배열 원소 제거하기 : var arrays = ["산업", "패션", "시각"]; arrays.pop() 해당 배열의 마지막 원소가 제거된다.

.concat - 배열 두 개를 결합해서 하나의 배열로 만들기 : var arrayA = ["산업", "패션", 시각"]; var arrayB = ["디자이너, 개발자"]; var arrayAB = arrayA.concat(arrayB); arrayA뒤에 arrayB가 이어진다. 같은 방법으로 여러 배열을 더할 수 있다. arrayA.concat(arrayB, arrayC);

.indexOf - 배열원소 색인 찾기 : var arrayColor = ["red", "green", "blue]; arrayColor.indexOf("red"); //1 배열에 포함되어있지 않은 원소를 물어보면 '-1'이 반환 된다. arrayColor.indexOf("magenta"); //-1

.join - 배열에 포함된 모든 원소를 하나의 문자열로 만들기 : var arrayColors = ["red", "green", "blue"]; arrayColors.join(); // "red,green,blue" .join(seperator) - 배열에 포함된 모든 원소를 구분자로 나눠진 문자열로 만들기 : arrayColors.join("-"); //"red-green-blue";

의사결정기

Math.random() :이 메서드를 사용하면 호출할 때마다 0과 1사이의 숫자하나를 무작위로 선택해서 반환. Math.random(); //0.89823932904223232 Math.random() * 10 : 0~과 10사이의 랜덤 숫자를 원할 때.

Math.floor() - 소수점 이하 버리고 정수로 만들기 : Math.floor(3.74); //3
