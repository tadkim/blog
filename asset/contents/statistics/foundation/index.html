확률과 통계

한국국토정보공사 LX의 확률과 통계 강의 및 문제 풀이를 보고 기초공부를 학습하였다. 실제적으로 응용할 것은 통계 파트이지만, 통계를 다루는 데 기본이 되는 것이 확률이므로 같이 학습을 진행하였다.

확률

시행과 사건

하나의 주사위를 던졌을 때 나오는 눈을 원소로 하는 집합 S={1,2,3,4,5,6}이 있을 때 S의 부분집합의 갯수는 2

통계

기술통계와 추론통계

  • 기술통계 (descriptive statistics): 자료를 수집하여 표나 그림으로 나타내거나 대표값, 분산도, 분포 형태 등을 도출하여 자료의 특성을 쉽게 파악할 수 있도록 자료를 정리하고 요약하는 통계학 분야
  • 추론통계 (inferential statistics): 표본에서 산출한 통계량을 근거로 모집단의 모수를 추론하는 통계학 분야. (전체 집단에서 구하는 것이 시간이나 비용이 많이 들기 때문에 표본을 추출해서 집단의 특징을 구하는 방법을 택함.)

기술통계의 개념

좋은 데이터가 수집되었다고 해도 정보가 명확하게 이해될 수 있게끔 정리되지 않으면 유용성이 떨어진다. 사용자의 이해를 돕기 위해 데이터를 요약하고 시각적으로 보기 좋게 표현하는 것과 관련된 기법이 바로 기술통계학이다. 수집된 데이터의 집합의 특성 및 분포를 파악하고, 그 분포특성을 요약적으로 설명해주는 대표치를 개발하려는 통계학의 분야이다.

기술통계 분석의 종류

도수분포표

도수

도수란 각 범주에 속하는 데이터의 개수이다.
  • 범주 : 계급 또는 구간
  • 도수분포표 : 분류된 구간별로 도수를 대응시켜 놓은 표
  • 상대도수 : 전체 도수의 합을 1로 했을 때 해당 구간의 도수의 상대치

등간척도나 비율척도로 측정된 데이터 범주화 방법

  1. 최대값과 최소값을 확인
  2. 최대값과 최소값의 차(범위)를 구함
  3. 전체 개수에 따라 구간 개수를 정함
  4. 범위를 구간의 개수로 나누어 구간의 폭을 정함
  5. 각 구간의 도수를 정함 (상대도수를 구하기도 함)

막대그래프와 원그래프

범주형 데이터로 된 도수분포표를 그림으로 나타낸 것이다.

  • 막대그래프 : 각 범주의 도수를 직접 비교
  • 원그래프 : 원을 1로 보고 각 범주의 도수를 전체 도수로 나눈 상대도수로 표현

히스토그램과 꺾은선 그래프

  • 히스토그램 : 양적 데이터로 된 도수분포표를 그림으로 나타낼 때 방법으로 막대 간에 간격이 없다는 점이 막대그래프와의 차이점이다.
  • 꺾은선 그래프 : 도수를 선으로 표시. 데이터가 연속적 성격을 가졌을 때 활용한다. 특히, 시계열 데이터를 나타낼 때 자주 활용한다.

줄기-잎 그림

데이터들의 배열을 이용하여 막대그래프와 유사한 효과를 낸다. 데이터의 수가 적은 경우에 효율적이며 막대그래프보다 많은 정보를 담고있다. 각 구간 내의 관측치뿐만 아니라 각각 관측치들이 얼마인지 보여준다.

특성치를 이용한 데이터 요약

기술통계학에서 사용되는 기술통계량(Descriptive Statistic)은 앞에서 설명한 것처럼 자료의 정보를 표시해주는 척도로 크게 3가지의 부류로 분류될 수 있다.

  • 자료의 중심위치에 대한 정보 제공 (대표값) : 자료들이 어떤 값을 중심으로 분포되었나?
  • 자료의 퍼진 정도에 대한 정보 제공 (산포값) : 자료가 어떤 형태로 분포되어 있으며, 중심(대표값)으로부터 얼마나 퍼져있는가?
  • 자료의 비대칭 정도 정보 제공 (비대칭도) : 자료의 분포가 기울어진 방향과 정도

대표값 (central tendency)

자료 전체의 특징을 표현하는 하나의 대표적인 수로써 자료의 집중 경향을 나타낸다. (산술)평균, 중앙값, 최빈값 등이 있다.

분산도(산포도, Measures of Dispersion or Variation)

변량들이 대표값으로부터 흩어져 있는 정도를 하나의 수로 나타낸 값. 즉 자료가 퍼져있는 정도. 분산도에는 여러가지 종류가 있는데, 표준편차, 평균편차, 사분편차, 범위, 변동 계수 등이 있다.

분포 형태 (shape)

그래프를 나타냈을 때 좌우로 어느정도 비대칭적인지(왜곡도), 또는 수직으로 얼마나 뾰족하게 솟아있는 지(첨도) 정도를 나타낸 지표이다. 왜곡도, 첨도가 있다.

대표값과 평균

분포가 대칭적일 수록 평균, 중앙값, 최빈값이 비슷하게 나타남.

평균 (Men)

n개의 변량(자료)의 평균(mean) 즉, 모든 자료의 합을 자료의 개수로 나눈 것이다. 계산이 쉽고 각 자료에 있어서 유일하게 구해진다는 장점이 있다. 하지만 극단적으로 큰 값이 있을 경우 평균을 대표값으로 사용하기에는 부적합하다. 평균은 극단값에 민감한 수치로, 몇 개의 극단값들만 있어도 표본의 크기가 작을 경우에는 대표값으로서의 기능을 상실한다.

중앙값 (median)

변량 전체를 크기 순서대로 나열할 때 중앙(50%)에 위치하는 변량의 값. 평균이 극단값에 크게 영향을 받는 것과는 달리, 중앙값은 극단값에 거의 영향을 받지 않는 통계량이다. 따라서 관측값의 수가 적고 극단값이 존재하는 경우에 대표값으로 적당하다. 중앙값은 수량으로 관측된 자료에만 이용이 가능하다.

표본의 중앙값을 구하는 식은 전체 변량의 개수가 홀수이냐 짝수이냐에 따라 다르다.

  • n이 홀수일 때, 중앙값은 𝑥 (n/2+1)
  • n이 홀수일 때, 중앙값은 { 𝑥 (n/2) + 𝑥 (n/2+1)} / 2

따라서, 변량(11, 9, 42, 25, 97, 40, 88, 75, 93)의 중앙값은 42가 되고, 자료집합 {5,6,7,8}의 중앙값은 6.5이 되는 것이다.

최빈값 (mode)

자료에서 가장 자주 나타나고 있는 or 관측빈도의 수가 가장 많은 관측치 또는 값. 질적 자료나 양적 자료 모두에 사용되며 자료의 분포에서 빈도수가 어느 곳에 가장 많이 밀집되어 있는가를 측정하는 것이다. 자료에 따라 존재하지 않을 수도 있으며, 또한 존재하더라도 유일하지 않을 수도 있다.
ex) 변량 (12, 9, 12, 15, 17, 15, 18, 15, 13)의 최빈값은 15

분산도와 표준편차

분산도는 자료의 대표값으로 자료의 중심위치는 측정할 수 있으나, 자료들이 어떠한 형태로 분포되어 있는지는 알 수 없다. 서로 다른 두 자료가 대표값은 같더라도 자료의 분포 형태는 매우 다를 수 있다. 따라서 자료의 분포에 대한 특성을 알아보기 위해서는 중심위치와 더불어 각 자료값들이 평균과 같이 대표값 주위에 흩어져 있는 정도를 측정하는 산포도에 대한 추가적인 정보가 필요하다.

범위 (Range)

범위란 자료들이 퍼져있는 구간의 크기로 관측치 중 최대치와 최소치의 차이를 말한다. 범위(R)은 다음과 같이 정의된다. R = 최대값 - 최소값. 범위는 극단값에 영향을 매우 많이 받으므로 불안정하다. 또한 범위는 일반적으로 많은 정보를 제공하는 척도는 아니며, 특히 자료의 크기가 큰 경우에 더욱 그렇다.

사분위편차 (Quartile Deviation; 사분위 범위수, Interquartile Range)

사분위편차란 자료들이 얼마나 중간부분에 집중되어 있는가를 나타내주는 퍼짐의 정도이다.

  • Q1 = 제1사분위수 (Lower Quartile) : 자료를 작은 것부터 크기 순서대로 나열하여 25%에 위치한 값
  • Q2 = 제2사분위수 or 중앙값(Median) : 자료를 작은 것부터 크기 순서대로 나열하여 50%에 위치한 값
  • Q3 = 제3사분위수 (Upper Quartile) : 자료를 작은 것부터 크기 순서대로 나열하여 75%에 위치한 값

분산 (variance)

편차(관측치와 평균의 차이)를 제곱하여 더한 후 이를 데이터의 개수로 나눈 값으로 데이터들이 중심 위치로부터 어느 정도 떨어져 있는가를 평균적으로 나타낸다.

표준편차 (standard deviation)

분산의 양의 제곱근. 표준편차는 각 변량이 평균으로부터 얼마나 떨어져 있는지를 측정하는 지표이다. 표준 편차가 작으면 평균값에 변량들이 밀집해 있는 것이고 표준 편차가 크면 그래프가 펑퍼짐한 모양, 즉 변량들이 넓게 분산되어 있음을 의미한다. 표준편차는 분산도를 나타내는데 가장 유용한 지표이다.

변동계수 (coefficient of variation, CV)

변동계수는 여러 종류의 자료의 산포도를 비교하는 데 사용되어진다. 자료값의 크기에 따른 상대적인 비교를 위하여 표준편차를 집단 평균으로 나눈 값이다. 따라서 표준편차의 산술평균에 대한 상대적 크기를 나타낸다. 서로 다른 평균과 표준편차를 갖는 여러 자료의 상대적인 변동 혹은 산포도를 측정하기 위해서는 각 자료의 평균과 표준편차를 동시에 고려한 변동계수가 유용하게 사용된다. 아래 예시를 보면 이해가 쉽다.

구분체중평균표준편차변동계수
남자체중72, 74, 77, 68, 66, 75724.2426410.058926
여자체중45, 48, 52, 53, 46, 50493.2249030.065814

위의 경우 남자 집단의 표준편차(4.243)가 여자집단의 표준편차(3.225)보다 더 크지만 이는 남자들의 체중이 원래 크기 때문이다. 사실상 여자들의 변동에 비해 남자들의 변동은 상대적으로 작다고 볼 수도 있는 것이다. (0.059 < 0.066)

기대값 (expected value)

기대값이란 모집단의 중심 즉, 확률분포의 무게 중심(균형점)을 말한다. 기대값을 구하는 방법은 각 확률변수에 확률을 곱한 것의 합이다. 확률변수가 이산형일 때의 기대값을 구하는 일반 공식은 다음과 같다.

E(𝑥) = 𝑥1p1+𝑥2p2+𝑥3p3+ ... + 𝑥npn = ∑𝑥ipi = ∑𝑥p(𝑥) = 𝛍

예를 들어 아래 확률 분포표에서 기대값을 구해보자.

x123
P(𝑥)2/51/52/5

E(𝑥) = 2/5 + (2 × 1/5) + (3 × 2/5) = 2

왜곡도와 첨도

왜곡도 (왜도, skewness)

도수분포의 좌우 비대칭 여부와 정도를 측정하는 지표 [오른쪽 / 왼쪽으로 치우침]. 왜곡도의 값은 -3에서 +3의 범위 값을 가진다.

  • 왜곡도 = 0 : 대칭분포 [평균=중앙값=최빈값]
  • 왜곡도 > 0 : 왼쪽으로 치우침 (positive skewed) 이때 최빈값 < 중앙값 < 평균.
  • 왜곡도 < 0 : 오른쪽으로 치우침, 왼쪽으로 꼬리가 길다. (negative skewed) 이때 평균 < 중앙값 < 최빈값.

첨도

첨도는 분포가 얼마나 중심에 집중되어있는지 뾰족함의 정도를 수량적으로 측정한 값이다.

  • 정규분포 (중첨) = 0
  • 정규분포보다 높은 봉우리, 뾰족한 모양 (급첨) > 0
  • 정규분포보다 낮은 봉우리, 무딘모양 (완첨) < 0
