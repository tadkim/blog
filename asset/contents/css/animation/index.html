Intro

이 책에서는 CSS transition, JS, SVG가 아니라 CSS 애니메이션 SPEC에 대해서만 다룬다.

A quick word on vendor prefixes

Vender prefix를 적용하지 않는다면 CSS animation을 제대로 구현할 수 없다. 책이 쓰여진 시점인 2013년에는 Firefox, Opera, IE의 최신 버전은 prefix없이도 CSS animation을 지원한다. 하지만 몇몇 브라우저나, 구형 브라우저들은 지원하지 않는 경우가 있으므로 작은 프로젝트라고 하더라도 prefix를 쓰는 것을 강력히 권장한다. (물론 내부적으로 실험하는 경우에는 사용하지 않아도 된다.) 책의 설명에서는 가독성을 위해 prefix를 생략했다.

What CSS to prefix? 사이트에서 반드시 prefix가 필요한 속성들이 분류되어 있으니 참고하도록 하자.

Keyframe

CSS animation을 정의하기 위해서는 @keyframes 규칙을 사용해야 한다. 규정할 애니메이션에는 이름을 부여해서 추후에 불러올 수 있게끔 해야 한다. 이런 이름은 동작의 특성으로 지정하는 게 좋다. 예를 들어 차(car)가 움직이는 애니메이션이라면 이름을 "drive"로 준다.
